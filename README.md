# Green Nest

A modern, responsive React web application for plant enthusiasts. Green Nest helps users explore plants, discover eco-decor ideas, and manage their profile with secure authentication.

## Features
- **Hero Slider:** Responsive slider showcasing plants with autoplay and timer.  
- **All Plants Page:** Browse all plants with a modern grid layout.  
- **Plant Details:** View individual plant details; access restricted to logged-in users.  
- **User Profile:** Protected page for users to view and edit their information.  
- **Authentication:** SignUp and SignIn pages using Firebase Authentication.  
- **Nested Routes:** Organized with `react-router` for better navigation.  
- **Private Routes:** Restricts access to protected pages for authenticated users.

## Tech Stack
- React
- React Router
- React Icons
- React Toastify
- Tailwind CSS
- DaisyUI
- Firebase Authentication
